不！查阅["React 中不使用 JSX"](https://reactjs.org/docs/react-without-jsx.html)了解更多。

### 我需要在 React 中使用 ES6(+) 吗？
不！查阅["React 中不使用 ES6"](https://reactjs.org/docs/react-without-es6.html)了解更多。

### 如何在 JSX 中写注释？

```jsx harmony
<div>
  {/* Comment goes here */}
  Hello, {name}!
</div>
```
```jsx harmony
<div>
  {/* It also works 
  for multi-line comments. */}
  Hello, {name}! 
</div>
```
