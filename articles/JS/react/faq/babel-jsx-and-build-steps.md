### 如何在 JSX 中写注释？

```jsx harmony
<div>
  {/* Comment goes here */}
  Hello, {name}!
</div>
```
```jsx harmony
<div>
  {/* It also works 
  for multi-line comments. */}
  Hello, {name}! 
</div>
```
